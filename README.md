# Image-to-Text-Symbol-Art
一个用来转换的小代码，把你的图片变成Text字符画，可以发steam评论区，在代码里插入安抚机魂，还可以用来给纯文字游戏做插图

如何使用 (Usage Guide)

Select Image: 点击 "Load Image"。\
Detail Mode (关键设置):\
Standard (推荐): 适合发给朋友或社交媒体，符号清晰。\
High Detail: 适合大尺寸图片或作为文档背景，符号极其丰富，远看像真图。\
Simple: 适合非常小的尺寸（如 50 宽），只勾勒大概轮廓。\
Contrast Boost: 如果生成的图片看起来灰蒙蒙的，向右拖动滑块增加对比度；如果黑死一片，向左拖动。\
Width:\
微信/手机浏览：建议宽度 30-50。\
电脑记事本：建议宽度 100-150。\
代码注释/文档：建议宽度 80。
